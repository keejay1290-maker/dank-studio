## Architecture summary

### Generator pipeline
builds.ts → App.tsx (sidebar) → generate(key, params) in generators/index.ts
  → generator function in generators/shapes.ts
  → Point3D[] using draw.ts helpers
  → applyLimit(1200) → Preview3D + export code

### Key constraints
- MAX_OBJECTS = 1200 — hard cap from DayZ server limit
- Param key in builds.ts MUST match p.xxx in generator exactly
- drawRing uses Math.floor() for panel count → scale≥1 → never gaps
- Sphere panels stored at y = surface - panelH/2 (pivot-bottom)
- Three.js rotation.y = +yawRad (NOT negative)
