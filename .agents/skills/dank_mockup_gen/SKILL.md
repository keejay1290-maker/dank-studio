# 🎨 Dank Mockup Gen: Visual Fidelity Prediction

This skill utilizes the `generate_image` tool to predict the 'S-Tier' visual outcome of a geometric masterpiece.

## 📝 Usage Protocol:
1.  **Analyze**: Review the `Point3D[]` output of a masterpiece generator.
2.  **Describe**: Synthesize the structural highlights (e.g., "A weathered 120m limestone tower with a golden apex, perched on a rock cliff").
3.  **Render**: Use `generate_image` with the prompt prefix "High-fidelity DayZ building preview: [Description], cinematic lighting, 4k, architectural photography."

## 🎯 Purpose:
- Providing the user with a "Vision" before they export to the game.
- Verifying thematic consistency (e.g., ensuring a Medieval build doesn't look like Scifi).
